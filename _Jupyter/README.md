## Introduction

## 安装运行

- 安装
	```
		pip install jupyter		// 会被安装到全局site-packages目录
	```

- 运行
	+
		```
			jupyter notebook
		```
	+ 若遇到`RuntimeError: No supported color terminal library`,[参考这个链接](https://stackoverflow.com/questions/43443974/no-supported-color-terminal-library-python-jupyter)
